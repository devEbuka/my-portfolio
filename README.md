# My DevOps Portfolio

A personal portfolio site built and deployed with a CI/CD pipeline.

## Live Site
[devebuka.github.io/my-portfolio](https://devebuka.github.io/my-portfolio)

## How it works
Every push to `main` automatically triggers a GitHub Actions workflow that deploys the updated site to GitHub Pages. No manual deployment steps.

## Stack
- HTML/CSS (static site)
- GitHub Actions (CI/CD)
- GitHub Pages (hosting)
